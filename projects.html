Influencer Marketing Web Application

As a Software Engineering Intern at StoryIt, I worked on developing an influencer marketing platform designed to streamline collaboration between businesses and content creators. I Miss My DJ - Interactive Music Experience

As part of Women in Computing at UCSD, I contributed to the development of "I Miss My DJ," an interactive web application that recreates the experience of listening to curated music playlists based on user nostalgia and personal preferences. This project aimed to bring a personalized, immersive music experience to users by simulating DJ-style music transitions and interactive audio controls.

  • Implemented real-time audio processing features using the Web Audio API
  • Developed user preference algorithms to create personalized playlist recommendations
  • Collaborated with a team of women engineers to design and implement the user interface
  • Integrated with music streaming services to access and manipulate audio content

Other Projects

  1. Brain-Computer Interface (BCI) Dino Game

    One of my most exciting projects involved integrating brain-computer interface (BCI) technology with the classic Google Chrome Dinosaur Game. Using the Muse Headband API, which records real-time EEG (electroencephalography) signals, I developed a system where a player could control the game by simply blinking. This project required extensive signal processing, as I had to filter and interpret EEG data to detect blinks and translate them into game inputs. I implemented a JavaScript-based interface to interact with the Muse API, allowing real-time response to brain activity. This project placed second in a district STEM competition, where I presented my findings to a panel of judges.
